What is HTML and it's basic structure

HTML stands for hypertext markup language. It is the standard markup language for creating web applications and web apps. It basically tells the browser what to display content and the type of that content with different tags.

Today I have learned many of those tags and how to use them for which purpose. So lets check out few of those.

Headings

There are six levels of heading. From h1 to h6.
I have started this code with h1, which is the biggest one and the size keeps decreasing all the way to h6.
We should always use one h1 for one page, use then use h2 and h3 under h2 and so on.
This is for better SEO and accessibility.

Paragraphs

We use <p> tag for paragraphs. It is a block level element and it creates a new line after the end of the paragraph.
We can also use <br /> tag for line break without creating a new paragraph.

Text format

We can use <strong> tag to make text bold and <em> tag to make text italic.
However, we should use strong and em for importance and emphasis only, not for styling.
For styling, we should use CSS.
or there is also <b> and <i> tags for bold and italic respectively, but they are not recommended for use as they do not convey any meaning.

Links and navigation

It's mostly about <a> tags for creating links and <nav> tag for navigation.
We can also set class and id in a tag and use a tag to jump and navigate to that preferred destination.
For example, go to the beginning of the page.

There are other attributes we can use inside a tag. href, target, rel etc. href="links/id" where you want to link to and Target="_blank" to open link in new tab and rel="noopener noreferrer" for security reasons.
